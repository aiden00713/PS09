----

----

### PS090201
> 設計一個程式，該程式會依序讀入數個學生成績，包含學號、國文、英文成績(輸入學號0表示結束，至多讀入20個學生成績)，之後找出國文及英文成績的平均（計算到小數點下兩位四捨五入）。

### PS090202
> 設計一個程式，該程式會依序讀入數個學生成績，包含學號、國文、英文成績(輸入學號0表示結束，至多讀入20個學生成績)，之後再讀入及格的分數，最後找出國文及英文成績的不及格的人數。
